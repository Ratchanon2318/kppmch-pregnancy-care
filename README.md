# KPPMCH Pregnancy Care Platform
### Next.js • Google Apps Script • LINE Messaging API

Digital platform modernizing maternal care for **Kamphaeng Phet Community Municipality Hospital (KPPMCH)**. Focuses on digitizing patient tracking and streamlining appointment workflows.

## Core Features

* **Appointment System:** Digital booking to reduce clinic congestion.
* **Maternal Automation:** Pregnancy tracking via LINE Messaging API & GAS.
* **Clinic Management:** Integrated ANC and Family Planning schedules.
* **Optimized UI:** High-performance styles using CSS Modules for scoped styling.
